Available Commands

List of all safezone commands and their usage

/szcoords
Client

Get your current coordinates and copy them to clipboard

Usage: /szcoords
Copies coordinates in Lua vector format
/szdebug
Client

Toggle debug visualization mode for zones

Usage: /szdebug [mode | ID]
Arguments:
mode Optional: all, active, inactive
ID Optional: Zone ID to debug single zone
Example: /szdebug 3
Shows zone boundaries with purple markers
/szadmin
Client

Open the safezone admin panel

Usage: /szadmin
Requires admin permissions
/sztoggle
Client

Toggle marker visibility for a specific zone

Usage: /sztoggle <ID>
Arguments:
ID Required: ID of the zone
Example: /sztoggle 3
/debugrender
Client

Toggle render distance debug information

Usage: /debugrender
Shows zone distance calculations in console
/setrenderdist
Client

Set render distance for a specific zone

Usage: /setrenderdist <ID> <distance>
Arguments:
ID Required: ID of the zone
distance Required: Distance in meters (0-500)
Example: /setrenderdist 3 200
/listsafezones
Client

List all safezones with detailed info in F8 console

Usage: /listsafezones [filter]
Arguments:
filter all (default), config, custom
Example: /listsafezones custom
